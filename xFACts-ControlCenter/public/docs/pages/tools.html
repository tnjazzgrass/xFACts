Tools

Shared operational tools and vendor specification catalogs
The Tools module houses shared infrastructure that powers operational actions across xFACts — BDL imports, payment processing, DM API operations, and the vendor specification catalogs that drive them all. If monitoring modules are the eyes and ears, Tools is the hands.
